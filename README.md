# Fruit Store
Install
-------

```$ npm install```

```$ npm install -g gulp```

How to run
-------

```$ gulp```

This will spawn a local server pointing to [`http://localhost:3002/`](http://localhost:3002/) and API server pointing to http://ec2-52-77-84-179.ap-southeast-1.compute.amazonaws.com:3000/ (json-server)
